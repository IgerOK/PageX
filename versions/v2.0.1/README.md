# PageX v2.0.1
>Интерфейс и функции v2.0 сохранены
## Что нового:
- При закрытии программа сворачивается в трей
- Закрыть программу можно из меню в трее
- Добавлена горячая клавиша Ctrl+F1 для сворачивания/разворачивания окна
- Ctrl+F1 работают глобально
- Исправлена нумерация строк при изменении шрифта
- Улучшена совместимость с разными версиями Windows

## Требования:
- AutoHotkey v2.0+
- Windows 10/11

## Основные горячие клавиши:
- Ctrl+F1 - Свернуть/развернуть окно (работает глобально)
- Ctrl+S - Сохранить
- Ctrl+Q - Сохранить и выйти
- Ctrl+1..0 - Переключение вкладок 1-10
- F1 - Справка
